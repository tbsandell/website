List

Operations

Name Types Syntax Details Example
List literal [any, any, ..., any] [e_1, e_2, ..., e_n] Returns a list with the objects e_1 through e_n ['hello', 2, 'u']
Access list[int] l[i] Returns the element at index i myList[3]
Negative Access list[int] l[-i] Equivalent to l[len(l)-i] myList[-2]
Slice list[int:int] l[i:j] Returns the elements at indexes i through j myList[2:4]
Slice with skip list[int:int:int] l[i:j:k] Returns the elements at indexes i + kn until j myList[3:7:2]
Add list + list myList1 + myList2 Returns the union of two lists [1,7,4] + [2,7]
Multiply list * int myList * i Returns myList + myList + ... i times [1,6,3] * 3
